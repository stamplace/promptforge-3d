# PromptForge 3D

Write a game. Play it now.

PromptForge 3D is a local-first mobile web app that turns user words into an instant playable 3D game.

## Product Rule

The user does not receive files.
The user does not receive a project.
The user writes words — and the game opens immediately.

## First Target

Prompt → Game DNA → Live 3D scene → Play now.
